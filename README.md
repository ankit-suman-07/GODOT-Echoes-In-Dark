# Echoes in the Dark

*A 2D top-down horror game built with Godot.*

> **Status:** 🚧 In Development

## Overview

Echoes in the Dark is a short horror game where the player explores an abandoned facility, manages limited resources, avoids a hostile creature, and uncovers the story through environmental exploration.

This project is primarily a learning and portfolio project focused on game programming, AI systems, and game design.

---

## Features

* Top-down player movement
* Flashlight system
* Enemy AI (Patrol → Chase)
* Line of Sight detection
* Key and door progression
* Story notes
* Ambient audio
* Safe room
* Win/Lose conditions
